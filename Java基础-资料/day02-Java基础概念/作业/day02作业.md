~~~java
// 数据类型 变量名 = 初始化值；
String str = "中国";
System.out.println(str);
// 其中String为数据类型，str为变量名（符合标识符规则即可），"中国"是我们学习的字符串常量值。
// 总体即表示：将字符串"abc"，赋值给字符串类型的变量str。
// 输出结果为：中国
~~~
